Overview

This project focuses on Feature Engineering, Model Optimization, and Performance Comparison using machine learning techniques on the California Housing Dataset. The main objective is to build regression models that can predict house prices accurately based on different housing-related features such as median income, house age, average rooms, population, and location.

The project begins with data loading and preprocessing, where the dataset is inspected for structure, missing values, and overall quality. Feature engineering is then performed by creating additional useful features to improve model performance. Feature scaling is applied to normalize the data and ensure better model training.

Three regression models are implemented and compared:

- Linear Regression
- Ridge Regression
- Decision Tree Regressor

The performance of each model is evaluated using important regression metrics such as RMSE (Root Mean Squared Error) and R² Score. These metrics help measure prediction accuracy and compare how well each model performs.

Finally, visualization techniques such as Actual vs Predicted plots are used to understand model behavior and prediction quality. Based on performance comparison, the best-performing model is identified for house price prediction.
